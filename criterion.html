NC curves.

acoustics.criterion.nc_curve(nc)[source]¶

Return an array containing the nc curve.

Parameter:

nc: int between 15 and 70 with step of 5. Valid values are: 15, 20, 25, ..., 60, 65 and 70. Invalid values (e.g. 23) returns None.

acoustics.criterion.nc(levels)[source]¶

It returns the NC curve of levels. If levels is upper than NC-70 returns ‘70+’.

Parameter:

levels: 1-D NumPy array containing values between 63 Hz and 8 kHz in octave bands.
